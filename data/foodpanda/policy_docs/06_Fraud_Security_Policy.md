# Fraud Prevention & Order Verification
**Objective:** Handling sensitive security escalations.

### Flagging Criteria
Orders over **$1,000** with differing Billing and Shipping addresses are automatically held for "Manual Review."

### Verification Protocol
Customers must confirm the last 4 digits of the card used and the exact billing address via a secure link provided by the agent.

### Agent Safety Constraint
**CRITICAL:** Agents are strictly forbidden from asking for a full Credit Card number or CVV code. If a customer provides this in chat, the agent must redact the text and escalate to a Lead Supervisor immediately.
